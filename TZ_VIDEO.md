# Техническое задание: Поддержка воспроизведения видео в GachiBass Radio

## 1. Общая цель
Реализовать возможность просмотра видеофайлов непосредственно на сайте без их копирования в директорию проекта, используя прямые ссылки на локальные файлы в файловой системе.

## 2. Функциональные требования

### 2.1. Интеграция медиа-контента
- **Источник видео:** Папка `Z:\гачибасс\GachiBassRadioV2\video` и корень `Z:\гачибасс\GachiBassRadioV2\`.
- **Метод привязки:** Использование абсолютных путей к файлам. Копирование файлов в папку сайта запрещено.
- **Формат данных:** Список видео должен храниться в массиве `GACHI_DATA` в файле `js/content.js` в категории `videos`.

### 2.2. Воспроизведение (Плеер)
- **Интерфейс:** Видео должно открываться в модальном окне (поверх основного контента).
- **Техническая реализация:**
    - При клике на кнопку «Смотреть ♂» вызывает функция `openPlayer`.
    - Внутри модального окна динамически создается элемент `<video>`.
    - Атрибуты элемента: `controls = true`, `autoplay = true`.
    - Путь к файлу должен приводиться к формату `file:///` для корректного открытия браузером.

### 2.3. Обработка путей
- **Windows-совместимость:** Все обратные слеши (`\`) в путях должны заменяться на прямые (`/`) перед передачей в `src` элемента.
- **Протокол:** Если путь не начинается с `file:///`, он должен быть добавлен автоматически.

## 3. Технические особенности и ограничения
- **Безопасность браузера:** Из-за ограничений CORS и политик безопасности браузеров, воспроизведение локальных файлов через `file:///` будет работать только в том случае, если сам `index.html` открыт как локальный файл (через проводник), а не через локальный сервер (http://localhost).
- **UI/UX:**
    - Список видео отображается в виде сетки (grid) из карточек.
    - Каждая карточка содержит название видео и кнопку запуска.
    - Модальное окно должно иметь кнопку закрытия и закрываться при клике вне области контента.

## 4. Структура данных (Пример в content.js)
```javascript
{
  "category": "videos",
  "items": [
    {"name": "Название видео", "path": "Z:/гачибасс/GachiBassRadioV2/video/file.mp4"},
  ]
}
```
